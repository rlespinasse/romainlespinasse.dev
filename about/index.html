    Opensource

    This blog is itself a project based on hugo with some github actions to automate it.

    In term of automation, I play with Github Actions

    • to ease the release of a repository : release-that ,
    • to slug some environment variables : github-slug-action ,
    • to slug any variable : slugify-value ,
    • to short some git references : shortify-git-revision ,
    • to expose some git commit data : git-commit-data-action ,
    • to export Draw.io diagrams : drawio-export-action .

    And a group of repositories dedicated to Draw.io diagrams exports

    • docker-drawio-desktop-headless : Docker image to serve Draw.io Desktop in a headless version.
    • drawio-exporter : Export Draw.io diagrams using command line,
    • drawio-export : Export Draw.io diagrams using docker,
    • drawio-export-action : Export Draw.io diagrams within GitHub Actions Workflow,

    In the rust world,

    • What I Need To See (wints ), a fuzzy term-based URLs opener,
    • drawio-exporter to export Draw.io diagrams using command line.

    And all my current and past projects are available on https://github.com/rlespinasse .

    You can also sponsor me on GitHub if you want.
